FAQ

Is this legal/compliance advice?

No. We help assemble/standardise inputs and produce a consistent reporting pack. You (and your counsel/auditor, if any) still approve and submit.

What standards do you support?

We work inside the framework you already use. Common UK/EU contexts include SECR, TCFD‑aligned disclosures, ISSB/IFRS S1/S2, and CSRD‑style questionnaires. If you’re unsure, we start with your existing outline and make the workflow consistent.

Do you need access to our systems?

Not initially. We can start with exports and shared files. If it’s a fit, we can discuss read‑only access later.

Turnaround?

Depends on scope and input maturity; Phase 1 aims to prove we can deliver a predictable cadence.

Pricing?

Concierge pricing in Phase 1 (fixed per monthly pack once scope is clear, in GBP). Founding member pricing for the first few teams.
