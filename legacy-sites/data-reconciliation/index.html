FAQ

Do you need access to our ad accounts?

Not to start. CSV exports are enough for the first pass. If it’s a fit, we can add read-only access later.

Which platforms do you support?

We start with 2–3 sources you already export (Meta, Google, TikTok, LinkedIn, GA4, Shopify). We expand only after the first workflow is stable.

What do you need from me?

Your exports + your “source of truth” definition (what number must match for billing/reporting).

What causes mismatches?

Common culprits: timezones, currency, attribution windows, tracking changes, campaign naming drift, duplicated lines, missing IDs, and backfilled conversions.

How fast is turnaround?

For a first pass: usually 24–48 hours after you send exports (depends on complexity).

Pricing?

Phase 1 is concierge: per reconciliation (weekly/monthly) with a simple fixed price once scope is clear.

How do you handle data / confidentiality?

Start with CSV exports only — no ad-account access required. If your exports include customer-level data, you can redact it first; we only need aggregate numbers + join keys (e.g., campaign/ad IDs, order IDs if relevant). If it’s a fit, we can work under an NDA and agree a simple retention/deletion approach.
